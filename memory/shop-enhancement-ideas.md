# 商店页面改进想法

**Why:** 商店是面向"购买者"的视角，用户不知道食物属性，应该在商品卡片上直接展示更多属性（饱腹值、经验值等），而不是只靠悬停 tooltip 才看到。

**How to apply:** 后续迭代时考虑：
- 商品卡片增加属性行（如 `🍽+20 ⭐+10`）
- 购买按钮上方或旁边展示关键属性
- 可考虑在卡片内折叠/展开详细信息
- 参考主流游戏商店的展示方式（如背包道具详情面板）
