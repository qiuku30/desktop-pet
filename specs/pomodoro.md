# 番茄钟需求说明

> ARCH-06 讨论确认 | 2026-07-16
> ⚠️ 不确定的地方必须问用户，不要猜测。

## 概述

主进程常驻的番茄钟计时器。25 分钟专注 + 5 分钟休息循环。通过系统通知、宠物浮动图标和气泡提醒用户。

## 核心行为

- 主进程 `setInterval` 每秒 tick，面板开关不影响计时
- 专注 → 休息 → 专注 → ... 无限循环
- 退出 App 直接重置，下次启动需用户手动开始
- 首期不做长休息

## 入口

| 入口 | 说明 |
|------|------|
| 面板导航栏新增"🍅 番茄"页 | 主力操作界面 |
| 宠物右键菜单 | 快捷启动/控制 |

## 计时参数

| 参数 | 默认 | 范围 | 说明 |
|------|------|------|------|
| 专注时长 | 25 分钟 | 5~120 | 用户可调，输入框 |
| 休息时长 | 5 分钟 | 1~60 | 用户可调，输入框 |

## 宠物态表现

| 状态 | 浮动图标 | 说明 |
|------|---------|------|
| 未开始 | 无 | 正常宠物 |
| 专注中 | `🍅 MM:SS` | 浮在宠物上方，持续显示 |
| 休息中 | `☕ MM:SS` | 同上 |
| 暂停中 | `⏸ MM:SS`（倒计时冻结） | 暂停中不区分专注/休息 |

## 番茄页布局（由上到下）

1. **进度环**：SVG 环形进度条，中间显示剩余时间 `MM:SS`
2. **状态文字**："专注中" / "休息中"
3. **操作按钮**：⏸ 暂停/▶ 继续、⏭ 跳过、✕ 放弃（或结束）
4. **统计**：
   - 🍅 今日完成数
   - 📊 总计完成数
   - 🔥 连续天数
5. **设置**：
   - 专注时长（分钟输入框）
   - 休息时长（分钟输入框）

## 右键菜单

### 未开始时
```
🍅 开始番茄
（分隔线，以下为原有菜单项）
```

### 专注中
```
🍅 专注中          → 点击切到面板番茄页
⏸ 暂停 / ▶ 继续
⏭ 跳过（去休息）
❌ 放弃
```

### 休息中
```
☕ 休息中          → 点击切到面板番茄页
⏸ 暂停 / ▶ 继续
⏭ 跳过（去专注）
❌ 结束
```

- 菜单项不显示倒计时（倒计时浮动图标已有）

## 操作逻辑

| 操作 | 状态 | 结果 |
|------|------|------|
| 暂停 | 进行中 | 计时器停，浮动图标变 ⏸ |
| 继续 | 暂停中 | 计时器恢复 |
| 跳过 | 专注中 | 进入休息，本次专注**不计数** |
| 跳过 | 休息中 | 直接进入下一个专注 |
| 放弃 | 专注中 | 不计数，重置为未开始 |
| 结束 | 休息中 | 计一个番茄，进入下一个专注 |
| 退出 App | 任意 | 重置为未开始 |

## 通知

- 专注结束 → 系统通知 `🍅 专注完成！` + 宠物气泡 `该休息一下啦～☕`
- 休息结束 → 系统通知 `☕ 休息结束` + 宠物气泡 `继续加油！💪`
- 主进程 `new Notification()`，静默失败不报错

## 统计

- 🍅 今日：N 次 · Xh Ym — 按本地日期，次数+时长，过日重置
- 📊 总计：N 次 · Xh Ym — 所有完成番茄的总次数+总时长
- 🔥 连续：连续完成番茄的天数（断了重置）
  - 同一天多次完成 → streak 不变
  - 昨天有完成 → streak + 1
  - 昨天无完成 → streak 重置为 1

统计面板显示"次数 + 时长"，热力图/周月年统计后续迭代。
