# 宠物系统需求说明

> ⚠️ 不确定的地方必须问用户，不要猜测。

## 概述

桌面悬浮宠物，能与用户互动，通过喂食和游戏获得成长。

## Phase 1 功能

### 悬浮窗口
- 透明无边框窗口，始终置顶
- Emoji 占位外观（后续替换美术资源）
- 闲置动画（眨眼、晃动等）
- 用户可拖拽移动
- 随机走动（基础移动，窗口边框攀爬 Phase 2 再做）

### 对话气泡
- 单击宠物弹出对话气泡
- 内容根据宠物心情和等级变化
- 连续点击叠加显示多个气泡
- 每个气泡 2 秒后自动消失
- 初始台词库 10~20 条

### 右键菜单
- 喂食：消耗食物，增加亲密度
- 状态：查看宠物当前属性
- 退出：关闭应用

### 面板切换
- 双击宠物 → 窗口变形展开为面板
- 面板包含：宠物状态页、导航栏、内容区
- 关闭面板 → 窗口缩回宠物态

### 赶跑
- 拖宠物到屏幕边缘 → 跑出屏幕
- 30秒~1分钟后自动回来

### 成长系统
- 亲密度数值（喂食增加）
- 等级（经验值累积升级）
- 外观进化：团子 → 长大 → 进化形态
- 升级解锁：新台词、新动作、服饰购买

## 验收标准
- [ ] 应用启动，宠物悬浮在桌面上
- [ ] 单击出气泡，多击叠加
- [ ] 双击切换面板态
- [ ] 右键菜单三项功能正常
- [ ] 可拖拽移动宠物
- [ ] 拖到边缘能赶跑并自动回来
- [ ] 关闭重开后状态恢复

## Phase 2：桌宠形象化

> ARCH-08 于 2026-07-24 讨论确认。技术设计见
> `docs/superpowers/specs/2026-07-24-pet-visualization-design.md`。

### 首发角色

- 原创幻想生物“奶油星团”，以圆润团子兽为基础物种
- 高清手绘 Q 版，不采用像素风、普通猫狗、Live2D 或 3D
- 暖色奶油主色，额头星纹和尾端星光作为少量星灵特征
- 首期只有一个基础形态，预留成长和进化形态

### 首期动画

- 待机
- 走路
- 进食
- 开心
- 低落
- 点击互动
- 睡觉

### 素材与表现

- 使用独立透明 PNG/WebP 动画帧和 JSON 皮肤清单
- 使用透明 Canvas 渲染，并适配高 DPI 和现有窗口缩放
- 面板角色卡显示当前皮肤立绘
- 现有点击、双击、拖拽、自动走动、喂食、气泡和番茄钟行为保持不变
- 皮肤与形态只改变表现，等级、经验、心情、饱腹、亲密度、金币和库存共享

### 本期不包含

- 多阶段进化素材
- 用户本地素材包导入
- 用户上传照片生成桌宠
- Live2D / Spine / 3D 渲染

### Phase 2 验收标准

- [ ] 启动后显示奶油星团，不再显示 Emoji 占位
- [ ] 七类动画按对应状态正确播放、打断和恢复
- [ ] 自动走动方向与动画朝向一致
- [ ] 单击、双击、拖拽、喂食、缩放和面板切换无功能退化
- [ ] 面板立绘与当前皮肤一致
- [ ] 高 DPI 下画面清晰，窗口尺寸不漂移
- [ ] 动作或素材加载失败时可回退，宠物不会白屏或消失
