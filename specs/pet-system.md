# 宠物系统需求说明

> ⚠️ 不确定的地方必须问用户，不要猜测。

## 概述

桌面悬浮宠物，能与用户互动，通过喂食和游戏获得成长。

## Phase 1 功能

### 悬浮窗口
- 透明无边框窗口，始终置顶
- Emoji 占位外观（后续替换美术资源）
- 闲置动画（眨眼、晃动等）
- 用户可拖拽移动
- 随机走动（基础移动，窗口边框攀爬 Phase 2 再做）

### 对话气泡
- 单击宠物弹出对话气泡
- 内容根据宠物心情和等级变化
- 连续点击叠加显示多个气泡
- 每个气泡 2 秒后自动消失
- 初始台词库 10~20 条

### 右键菜单
- 喂食：消耗食物，增加亲密度
- 状态：查看宠物当前属性
- 退出：关闭应用

### 面板切换
- 双击宠物 → 窗口变形展开为面板
- 面板包含：宠物状态页、导航栏、内容区
- 关闭面板 → 窗口缩回宠物态

### 赶跑
- 拖宠物到屏幕边缘 → 跑出屏幕
- 30秒~1分钟后自动回来

### 成长系统
- 亲密度数值（喂食增加）
- 等级（经验值累积升级）
- 外观进化：团子 → 长大 → 进化形态
- 升级解锁：新台词、新动作、服饰购买

## 验收标准
- [ ] 应用启动，宠物悬浮在桌面上
- [ ] 单击出气泡，多击叠加
- [ ] 双击切换面板态
- [ ] 右键菜单三项功能正常
- [ ] 可拖拽移动宠物
- [ ] 拖到边缘能赶跑并自动回来
- [ ] 关闭重开后状态恢复
